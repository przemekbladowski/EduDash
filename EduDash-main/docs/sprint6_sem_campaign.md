### 3.2 Kampania Display (Sieć reklamowa)

**Formaty reklam:**
- Responsive Display Ad (automatyczne dopasowanie)
- Banner 300×250 (medium rectangle)
- Banner 728×90 (leaderboard)
- Banner 320×50 (mobile leaderboard)

**Targetowanie:**
- Tematy: Edukacja, Zarządzanie oświatą, Szkolnictwo
- Grupy demograficzne: wiek 30–60, wyższe wykształcenie
- Niestandardowe grupy odbiorców: „Dyrektorzy szkół szukający oprogramowania"
- Remarketing: odwiedzający /landing, którzy nie wypełnili formularza

**Copy Display:**
```

```

## 5. Śledzenie konwersji (GA4 + GTM)

### 5.1 Zdarzenia GA4 zaimplementowane w kodzie

Plik: `src/app/utils/analytics.ts`

| Zdarzenie GA4 | Typ | Trigger | Wartość konwersji |
|---|---|---|---|
| `contact_form_submitted` | MAKRO-konwersja | Formularz na /landing → „Wyślij" | 150 zł (lead value) |
| `cta_clicked` | MIKRO-konwersja | Klik CTA „Wypróbuj" / „Umów demo" | 10 zł |
| `pwa_install_prompt_shown` | MIKRO-konwersja | Wyświetlenie promptu PWA | 5 zł |
| `app_installed` | MIKRO-konwersja | Instalacja PWA | 30 zł |
| `login_success` | MIKRO-konwersja | Udane logowanie demo | 20 zł |

### 5.2 Parametry UTM

Każdy URL docelowy zawiera parametry UTM:
```
utm_source=google
utm_medium=cpc
utm_campaign={nazwa_kampanii}
utm_content={wariant_reklamy}
utm_term={slowo_kluczowe}
```

### 5.3 Konfiguracja Google Tag Manager

```
Tag 1: GA4 – contact_form_submitted (Conversion)
  Trigger: Custom Event → contact_form_submitted
  Variables: user_role, conversion=true

Tag 2: GA4 – cta_clicked
  Trigger: Custom Event → cta_clicked
  Variables: cta_label, location

Tag 3: Google Ads Conversion Tracking
  Trigger: contact_form_submitted
  Conversion ID: AW-XXXXXXXXX/YYYYY
  Conversion Value: 150 PLN
```
